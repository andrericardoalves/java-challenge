### SENIOR LEVEL TEST
*15 minutes to solve*

Given an unsorted integer array nums, return the smallest missing positive integer.



Example 1:

```
Input: nums = [1,2,0]
Output: 3
```

Example 2:
```
Input: nums = [3,4,-1,1]
Output: 2
```

Example 3:
```
Input: nums = [7,8,9,11,12]
Output: 1
```



Constraints:

* 1 <= nums.length <= 5 * 10<sup>5</sup>

* -2<sup>31</sup> <= nums[i] <= 2<sup>31</sup>-1

* nums[] can have duplicate values
